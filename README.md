# Recursive Understanding Engine (RUE)

## Idea
An AI-powered system that explains concepts recursively.

##  How it works
- User asks question
- AI answers
- Extracts key concepts
- User clicks → deeper explanation

##  Tech Stack
- Python (Flask)
- Grok API
- HTML/CSS/JS

##  Run Locally
pip install -r requirements.txt
python app.py
